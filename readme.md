# Мини-проект: «Country Explorer»

 Country Explorer — это веб‑приложение, которое позволяет находить страны по названию, просматривать подробную информацию и загружать данные о соседних государствах. Проект построен на архитектуре из нескольких классов и использует публичный REST Countries API для получения актуальных данных о мире.

Приложение демонстрирует умение работать с внешними API, обрабатывать ошибки, структурировать код в виде сервисов и моделей, а также динамически обновлять интерфейс.
---

## Основные возможности.

1. Поиск страны по названию с автоподстановкой данных из REST Countries API.
2. Отображение детальной информации: флаг, официальное название, столица, население, регион, подрегион, площадь, валюты и языки.
3. Загрузка списка соседних стран по их кодам (CCA3).
4. Переход к соседним странам по клику на карточку.
5. Обработка ошибок (страна не найдена, пустой ввод, сетевые проблемы).
6. Индикация загрузки и аккуратное управление состоянием интерфейса.

## Использованные технологии
1. JavaScript (ES6+) — асинхронные запросы, классы, работа с промисами, обработка ошибок.
2. REST Countries API — получение данных о странах, соседях, валютах и языках.
3. ООП‑архитектура — разделение логики на классы ApiService, Country, CountryView, CountryService.
4. Работа с DOM — динамическая генерация карточек, обновление интерфейса, обработка событий.
5. Fetch API — сетевые запросы с async/await.
6. Intl.NumberFormat — форматирование чисел и локализация.
7. Чистая модульная структура — разделение ответственности, читаемость и расширяемость кода.
